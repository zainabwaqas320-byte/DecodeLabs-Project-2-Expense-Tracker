# DecodeLabs Project 2 - Expense Tracker

## Description
This repository contains my Project 2 (Expense Tracker) completed during the DecodeLabs Python Programming Internship.

The application allows users to record multiple expenses, calculates the total amount spent, validates user input, and displays a final expense summary.

## Technologies Used
- Python
- Google Colab

## Features
- Add multiple expenses
- Calculate total expenses
- Input validation using try-except
- Quit option to end the program
- Displays total transactions and total amount spent
